# Business-Intelligence-Stock-Analysis-
The project develops a system for real-time stock market data analysis and visualization using a scalable data streaming architecture. Live stock data is collected from the Yahoo Finance API and processed through a Java-based data ingestion component that acts as a Kafka producer. The data is then published to a Kafka topic for reliable data streaming. Apache Spark Streaming consumes the streaming data and performs financial analytics such as Moving Average, Volatility, and Trend analysis. The system also incorporates machine learning models such as Linear Regression and Random Forest for stock price prediction and analysis. The processed results are stored in a MySQL database for efficient data management. A Spring Boot framework is used to establish seamless communication between the frontend and backend, enabling efficient data exchange and system integration. Power BI is used to create interactive dashboards and reports for real-time financial insights. This architecture enables efficient processing of large volumes of stock data and supports data-driven investment decisions through clear data visualization, analytics, and improved prediction accuracy.
